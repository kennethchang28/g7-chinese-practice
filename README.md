# 國中七年級下學期・第三次段考 國文練習系統

一個**離線可用、題庫型**的國文練習網頁，仿**國中教育會考**素養導向命題（長題幹、情境化、含**閱讀題組**）。每按一次就抽出 **20 題選擇題**（單題與閱讀題組混合），全程**計時**，作答後**自動批改**並附**逐題詳解**；點「再出一份」會給你**全新題目**，且**保證不與上一份重複**。

> 🔗 **線上直接使用**：<https://kennethchang28.github.io/g7-chinese-practice/>
> （手機、平板、電腦的瀏覽器都能直接打開，免安裝。）

---

## 📚 範圍與難度

對應**翰林版**七年級下學期第三次段考，題庫共 **293 題**（含 14 組閱讀題組）：

| 課次 | 篇名 | 作者／出處 |
|------|------|-----------|
| 第七課 | 五柳先生傳 | 陶淵明（東晉） |
| 第八課 | 摩登土產鳳梨酥 | 洪愛珠《老派少女購物路線》 |
| 第九課 | 謝天 | 陳之藩《在春風裡》 |
| 第十課 | 貓的天堂 | 左拉（法國，翻譯文學） |
| 語文常識 | 漢字流變與書法欣賞 | — |
| 自學篇章 | 越南安妮 | 新住民主題 |
| 綜合測驗 | 字音字形・成語・修辭・標點・文學常識 | 跨課 |

**難度定位**：對齊國中教育會考中偏難，以「**會考考滿分**」為訓練目標。題目參考近十年會考出題風格——題幹具一定長度與深度（一般單題題幹 30 字以上，平均約 70～80 字），強調閱讀理解、推論、統整與評鑑。題型涵蓋：
- **閱讀題組**：文言對讀（五柳先生傳 × 愛蓮說／陋室銘）、古典詩（飲酒）、現代散文、新詩、生活應用文（公告／說明）、古今跨文本統整。
- **情境單題**：文言字詞、古今異義、成語語境、修辭、標點與語文應用、文學與文化常識。

## ✨ 特色

- **每份 20 題、單題＋閱讀題組混合**：依各課**分層抽樣**，閱讀題組整組呈現（一篇選文配多題），橫跨多課、題型多元。
- **計時器**：作答全程碼表計時，交卷後顯示用時。
- **再出一份不重複**：系統會記住已出過的題目，**連續兩份試卷絕不重複**（題庫足夠時可連續產生 14 份完全不重複的試卷）。
- **自動批改＋逐題詳解**：交卷後立即顯示分數、各課答對率，每題附正解與詳細說明。
- **純前端、可離線**：只有 HTML / JS，無需網路與安裝，資料保存在瀏覽器本機。

## 🚀 使用方式

**方法一（最簡單）**：直接打開上面的線上網址。

**方法二（下載到本機）**：下載本專案後，用瀏覽器打開 `index.html` 即可（需與 `quiz-core.js`、各 `questions_*.js` 放在同一資料夾）。

## 🧪 給想驗證／修改題目的人

```bash
# 需要 Node.js
node test_quiz.js     # 執行單元與整合測試（題庫結構、抽題防重複、批改、計時，應為「失敗：0」）
node server.js        # 本機啟動預覽伺服器 http://localhost:4322
```

- `index.html` — 介面（出題、計時、作答、批改、詳解、再出一份）
- `quiz-core.js` — 抽題／批改／計時引擎（瀏覽器與 Node 共用，純函式）
- `questions_*.js` — 分課題庫（L7、L8、L9、L10、語文常識、越南安妮、綜合）
- `questions.js` — Node 端題庫彙整（供測試使用）
- `test_quiz.js` — 單元與整合測試
- `server.js` — 本機預覽伺服器
